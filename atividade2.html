O que é Selection Sort

O Selection Sort é um algoritmo de ordenação que divide o vetor em duas partes: uma parte já ordenada e outra ainda não ordenada. A cada iteração, ele procura o menor elemento da parte não ordenada e o coloca na posição correta, realizando uma troca.

Exemplo passo a passo

VETOR INICIAL:

5 3 8 4 2

  1. Menor número é 2 -> troca com 5
  2. Depois procura o menor entre os restantes
  3. Repete até ordenar tudo

RESULTADO FINAL:

2 3 4 5 8

Tabela de Iterações

Iteração Vetor
inicial 5 3 8 4 2
1 2 3 8 4 5
2 2 3 8 4 5
3 2 3 4 8 5
4 2 3 4 5 8

Na segunda iteração, não houve troca porque o elemento na posição já era o menor da parte não ordenada. Mesmo assim, o algoritmo percorre o vetor para confirmar isso.

Código em C++

 
        #include <iostream>
using namespace std;

int main() {
    int vetor[5] = {5, 3, 8, 4, 2};
    int i, j, min, temp;

    for(i = 0; i < 4; i++) {
        min = i;

        for(j = i + 1; j < 5; j++) {
            if(vetor[j] < vetor[min]) {
                min = j;
            }
        }

        temp = vetor[i];
        vetor[i] = vetor[min];
        vetor[min] = temp;
    }

    for(i = 0; i < 5; i++) {
        cout << vetor[i] << " ";
    }

    return 0;
}

Explicação do código

cria um vetor de 5 posições do tipo inteiro
cria quatro variáveis:
"i" controla o primeiro laço (posição atual)
"j" percorre a parte não ordenada
"min" guarda o índice do menor elemento encontrado
"temp" é a variável temporária para fazer a troca

primeiro for:

Esse laço, controla as iterações do Selection Sort; Temos 5 elementos e precisamos de apenas 4 iterações (n-1); A cada volta, fixamos uma posição e vamos encontrar o menor valor para colocar nela
Depois assumimos que o menor elemento é o que está na posição atual


segundo for:

Percorre a parte ainda não ordenada, começa em "i+1" por conta que não precisamos comparar o elemento com ele mesmo.


comparação:

O elemento atual é menor do que eu conheço?
se for: atualizamos o índice do menor elemento

troca:

  1. Guarda o vetor do "vetor[i]" em "temp"
  2. Coloca o menor na posição "i"
  3. Coloca o valor antigo na posição do menor

sem "temp" você perderia um dos valores.

Imprimindo o vetor ordenado:

"for(i = 0; i < 5; i++)" percorre o vetor inteiro
"cout << vetor[i] << " ";" imprime cada número separado por espaço

Vantagens e Desvantagens

  • ✅fácil de entender
  • ✅fácil de implementar
  • ❌não é eficiente para grandes volumes de dados
  • ❌complexidade o(n²)
